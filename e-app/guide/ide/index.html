# 框架引入

  1. 将本小程序框架（源码）克隆至本地，并安装依赖。
git clone https://github.com/cjl255352/e-app.git
npm i
  1. 点击打开项目，选择框架
  2. 选择并确认项目类型
  3. 点击完成

# 关联应用

  1. 登录IDE工具
  2. 左侧选择企业内部应用
  3. 选择关联的应用（之前创建的应用）
