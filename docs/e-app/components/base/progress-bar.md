# progress-bar 滑块

## 个性属性

### step

- 类型: `number`
- 默认值: `1`

间隔。

### min

- 类型: `number`
- 默认值: `0`

最小值。

### max

- 类型: `number`
- 默认值: `0`

最大值。

### unit

- 类型: `string`
- 默认值: `'%'`

后缀。

### showValue

- 类型: `boolean`
- 默认值: `false`

是否在label后面接value加unit。

### handleColor

- 类型: `string`
- 默认值: `'#fff'`

滑块按钮的颜色。

### activeColor

- 类型: `string`
- 默认值: `'#108ee9'`

激活的拖动条颜色。

### backgroundColor

- 类型: `string`
- 默认值: `'#ddd'`

拖动条颜色。
