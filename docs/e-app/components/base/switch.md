# switch 切换

## 个性属性

### value

- 类型: `任意`
- 默认值: `false`

当前value，只读，设置无效。

### trueValue

- 类型: `任意`
- 默认值: `true`

勾选时的值。

### falseValue

- 类型: `任意`
- 默认值: `false`

不勾选时的值。

### trueLabel

- 类型: `string | boolean`
- 默认值: `true`

勾选时的标签。

### falseLabel

- 类型: `string | boolean`
- 默认值: `false`

不勾选时的标签。

### checked

- 类型: `boolean`
- 默认值: `false`

是否勾选。

### color

- 类型: `string`
- 默认值: `'#3296FA'`

勾选的颜色。

### showValue

- 类型: `boolean`
- 默认值: `false`

是否显示value。
