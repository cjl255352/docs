## 介绍

表单组件包含各种基础组件、子表。搭配[表单渲染函数](../../render/form)，可以快速生成一个表单页面，**不建议单独使用**。

## 属性

### bizObj

- 类型：`array`
- 默认值：空

基础组件数组，内为对象，表单组件遍历数组，根据对象生成对应的基础组件。

### btnPos

- 类型：`int`
- 默认值：`''`

表单动态按钮的位置，表单动态按钮会渲染在页面最下方。

### disabled

- 类型：`boolean`
- 默认值：false

是否禁用子表删除、子表新增、底部按钮。

### onRules

- 类型：`function`
- 默认值：()=>[]

返回校验方法的方法，返回一个数组，包含所有字段的校验方法（如果有定义），搭配表单渲染函数才有效。

### background

- 类型：`string`
- 默认值：`'#FFF'`

表单背景，参考css内background。
